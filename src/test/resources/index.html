Container

Container可以对对象进行实例化。比如： 


    Container container = new Container();
    container.addComponent(ArrayList.class);
    List list = container.getInstance(ArrayList.class);

    // Will equals to
    List list2 = new ArrayList()

简单依赖

在上述的例子中，ArrayList存在无参构造器，所以通过getInstance方法可以直接实例化。在大多数场景，对象是会存在依赖关系的。 比如： 


    public class Car {
        private final Engine engine;

        public Car(Engine engine) {
            this.engine = engine;
        }


        public Engine getEngine() {
            return engine;
        }
    }

在如上的例子中，Car对Engine有着依赖关系，那么通过Container获取Car的实例时，将会同时


    Container container = new Container();
    container.addComponent(Engine.class);
    container.addComponent(Car.class);

    Car car = container.getInstance(Car.class);
